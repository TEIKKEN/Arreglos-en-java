Proyecto Arreglos en Java
Objetivo del proyecto

Crear un programa en Java que utilice arreglos unidimensionales para almacenar números ingresados por el usuario, calcular la diferencia entre dos arreglos y mostrar el promedio de los resultados.
Instrucciones de ejecución
1) Abrir el proyecto en un IDE.
2) Ejecutar la clase Arreglos.java.
3) Ingresar los números solicitados por el programa.
4) El sistema mostrará los resultados y el promedio.

## Ejemplo de ejecución

### Entrada de datos:
El programa solicita al usuario 7 pares de números:

```
Ingrese el valor para el arreglo 1
5
Ingrese el valor para el arreglo 2
2
Ingrese el valor para el arreglo 1
10
Ingrese el valor para el arreglo 2
3
... (se repite hasta completar 7 pares)
```

### Salida esperada:
```
Los valores del arreglo 3 son: 
3
7
11
15
19
23
27

El promedio de la suma es: 15.142857142857142
```

## Descripción técnica

- **Arreglo 1:** Almacena los primeros 7 números ingresados
- **Arreglo 2:** Almacena los segundos 7 números ingresados  
- **Arreglo 3:** Contiene la diferencia (arreglo1 - arreglo2)
- **Promedio:** Se calcula como (suma de arreglo3) / 7
