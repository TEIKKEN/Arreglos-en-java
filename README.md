Proyecto Arreglos en Java
Objetivo del proyecto

Crear un programa en Java que utilice arreglos unidimensionales para almacenar números ingresados por el usuario, calcular la diferencia entre dos arreglos y mostrar el promedio de los resultados.
Instrucciones de ejecución
1) Abrir el proyecto en un IDE.
2) Ejecutar la clase Arreglos.java.
3) Ingresar los números solicitados por el programa.
4) El sistema mostrará los resultados y el promedio.
